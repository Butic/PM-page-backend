## Description

This is the repo of exceed-portal backend, this repo based on [Nest](https://github.com/nestjs/nest) framework.
before you start please check [Nest Documentation](https://github.com/nestjs/nest)

## Installation

```bash
$ npm install
```
```diff
- don't forget to move /config-example to /config
- the config example include a version of configs that will help you run your local env
- (optional) you can move .env-example to .env and add any env parameters to the file.
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
